Login Page With Password Checker

This password strength tester uses the zxcvbn API to check the strength of the password. The zxcvbn algorithm assigns a strength value of 0-4 to the password and updates the meter. All of the password-checking is done the user's end. I used JavaScript to check the password input field and to make the password checker only appear when there is an input and to display a qualitative value to the password.

George Washington Timeline

I wanted to create a dynamic timeline page that popped in when scrolling down. I started off by sketching out what I wanted it to look like on phones, tablets, and desktop viewports. Next, I found some cool George Washington photos and used Photoshop to optimize them for the most efficient asset delivery. I used jQuery to have it "pop in" on scroll. My favorite part is the tiny George Washington head at the center of each section.
